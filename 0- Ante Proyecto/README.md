## What is in the Proposal?

The proposal defines the scope of the thesis:

- **Problem statement**: Manual monitoring of *Aedes aegypti* mosquito populations is costly, slow, and imprecise. There is a need for an automated, intelligent system.
- **Proposed solution**: Design and build an IoT smart trap with computer vision capable of autonomously detecting, classifying (by sex), and reporting mosquito captures.
- **Objectives**:
  1. Develop an agent-based model (ABM) of mosquito population dynamics.
  2. Train a vision AI model to classify mosquitoes as male/female.
  3. Build a physical smart trap prototype.
  4. Deploy a web server to aggregate and visualize trap data.
- **Justification**: *Aedes aegypti* is the primary vector of dengue, Zika, and chikungunya — diseases of major public health concern in Paraguay.
