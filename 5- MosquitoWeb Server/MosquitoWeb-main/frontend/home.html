Aplicación de Deep Learning y Modelado Basado en Agentes para el Desarrollo de una Trampa Inteligente de Mosquitos

Una tesis que combina una trampa inteligente de clasificación de mosquitos y un modelo de simulación espacial, contribuyendo al control del dengue en el marco del Proyecto Wolbachia Paraguay.

Conocer el Proyecto Ver Modelo IA Ver Simulación Dashboard en Vivo

El problema: dengue en Paraguay

El Aedes aegypti es el principal vector de dengue, zika y chikungunya en Paraguay. El país sufre brotes anuales con cientos de miles de casos notificados, constituyendo una amenaza crítica para la salud pública. Las estrategias de control tradicionales son insuficientes para frenar la reemergencia sostenida del virus.

El Proyecto Wolbachia Paraguay busca liberar mosquitos portadores de la bacteria Wolbachia—que bloquea naturalmente la replicación viral—para reducir la transmisión de manera sostenible. Esta tesis aporta la infraestructura tecnológica de datos que dicho proyecto necesita para escalar.

Contribución de la Tesis

Dos aristas tecnológicas complementarias

Estimación de poblaciones y adquisición de datos de mosquitos son tareas extremadamente complejas. Esta tesis ataca el problema desde dos ángulos que se retroalimentan.

Arista I — Trampa Inteligente

Automatización y Adquisición de Datos

Se diseñó e implementó una trampa de captura con cámara embebida (Raspberry Pi 5) que ejecuta un pipeline de inferencia en tiempo real:

  • Detección: YOLOv8s entrenado con dataset propio (256×256)
  • Clasificación: MobileNetV3-Large exportado a ONNX
  • Telemetría: envío automático a PostgreSQL / Supabase
  • Sensores: temperatura y humedad (DHT22)
Arista II — Simulador ABM

Modelado y Simulación de Escenarios

Se implementó un modelo basado en agentes que simula la dispersión espacio-temporal de Ae. aegypti sobre grillas georreferenciadas derivadas de OpenStreetMap:

  • Entorno: grilla urbana con tipos de terreno (OSM + ESRI)
  • Agentes: estados J, M, FU, FG, D con dinámica estocástica
  • Movimiento: núcleo de Markov guiado por campo de potencial
  • Aplicación: análisis de escenarios para liberaciones Wolbachia
