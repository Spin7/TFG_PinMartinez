# References

This folder contains the **bibliography and reference literature** used throughout the thesis, organized by topic area.

## Folder Structure

```
References/
│
├── 1- Simulacion Agentes/          # ABM and agent-based modeling references
├── 2- Trampa/                      # Smart trap and IoT mosquito monitoring references
├── 3- Problema Epidemiologico/     # Epidemiological context (dengue, Aedes aegypti)
└── 4- FIUNA_TFG_linea_Mosquitos/   # Related FIUNA thesis line on mosquitoes
```

## Topic Areas

### 1 — Agent-Based Simulation (`1- Simulacion Agentes/`)

Key papers on ABM methodology and mosquito population modeling:

| Paper | Topic |
|-------|-------|
| `Adriano.2014.pdf` | Agent-based mosquito population dynamics |
| `HyunAutomatos.2008.pdf` | Cellular automata for biological simulations |
| `Ishida.2023.pdf` | ABM for urban vector control |
| `Jing Chen.2023.pdf` | Comprehensive mosquito behavior modeling |
| `ManeeratDaude.2017.pdf` | *Aedes aegypti* population dynamics model |
| `Yamashita.2018.pdf` | Urban environment mosquito simulation |
| `Libros/` | Reference books (complex systems, ABM theory) |

### 2 — Trap Design (`2- Trampa/`)
Papers and datasheets related to existing mosquito trap technologies, IoT sensor integration, and automated sorting mechanisms.

### 3 — Epidemiological Problem (`3- Problema Epidemiologico/`)
Literature on dengue fever in Paraguay, *Aedes aegypti* biology and ecology, and public health monitoring frameworks.

### 4 — FIUNA TFG Mosquito Research Line (`4- FIUNA_TFG_linea_Mosquitos/`)
Related thesis work and research from the FIUNA mosquito research line that provides institutional context for this project.
